 # Judocsarnok Mindössze hét hónap telt el az alapkőletétel óta, amikor 2016. június ötödikén átadták a kaposvári judocsarnokot.
 Magyarország legkorszerűbb dojója lett a Kaposvár, mely egyben a dél-dunántúli régió központja, és a kaposvári judo akadémia helyszíne is. Közel 180 millió forintból épült fel az ország legmodernebb judo létesítménye, amely háromszáz gyermek testi és lelki fejlődését hivatott biztosítani.
 A csarnokot természetesen judo bemutatóval avatták fel, melyet az óvodások kezdtek, majd a felnőtt korosztályig bezárólag mindenki tatamira lépett. A kaposvári cselgáncsozók sikertörténete az elmúlt években is tovább íródott, napjainkban pedig már a legkisebbek is közelebbről megismerkedhetnek ezzel a szép sportággal.
