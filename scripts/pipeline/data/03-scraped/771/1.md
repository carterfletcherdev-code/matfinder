##### PRIVATE LESSONS ## Accelerated Progression Via One On One Private Lessons If you would like to learn BJJ with maximum efficiency, private lessons are the answer!
 Group classes are the dominate way of learning jiu jitsu, however some people prefer to supplement their training with private lessons. Private lessons are an awesome way to speed up your personal development, as they allow for a more systematic, and one on one class setting.
