 PROGRAM QUESTIONS
 ## Which Program is Right for You? ### What's the difference between Gi and No-Gi BJJ? Gi uses the traditional uniform—grips, chokes, and control with the jacket and pants. No-Gi is faster, with shorts and a rash guard. Both teach the same core grappling, but Gi is more methodical while No-Gi feels more athletic.

 ### Which program should I start with as a beginner? Most beginners start with Gi BJJ. It's easier to learn control and technique when you can grip the uniform. But you can start with any program that interests you—we teach fundamentals in all classes.

 ### Can I train both BJJ and MMA at the same time? Yes. Adult Unlimited memberships include all classes—Gi, No-Gi, and MMA. Many students cross-train to build a complete skill set.

 ### What age does Kids BJJ start? Kids can start as young as 5-6 years old. We focus on age-appropriate instruction, keeping classes fun while building discipline and technique.

 ### Do I need my own gi or equipment to start? Not for your free week trial. Wear athletic clothes. After that, you'll need a gi for Gi classes ($60-120), and a rash guard and shorts for No-Gi ($40-80). We can help you get set up.

 ### How are private lessons structured? One-on-one sessions with a coach. You pick the focus—tighten fundamentals, prep for competition, or work on specific techniques. $50 per session. Just reach out to schedule.
