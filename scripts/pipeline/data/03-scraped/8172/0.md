#### ΟΜΑΔΙΚΑ ΤΜΗΜΑΤΑ

 Στη σχολή μας θα βρείτε ομαδικά τμήματα εξάσκησης Krav Maga, ενήλικων και παιδιών.
 Τα τμήματα των ενήλικων είναι χωρισμένα σε αρχάρια και προχωρημένα, ενώ ταυτόχρονα πραγματοποιούνται και μεικτές κοινές προπονήσεις. Έτσι, οι ασκούμενοι έχουν τη δυνατότητα να προπονηθούν στην κατάλληλη ύλη για το επίπεδό τους, αλλά και να λάβουν μέρος σε προπονήσεις με πιο προχωρημένες τεχνικές, οι οποίες διευρύνουν τους ορίζοντές τους και παρέχουν μια πιο γενική εικόνα για την αυτοάμυνα.
 Τα παιδικά τμήματα είναι χωρισμένα ανά ηλικιακή κατηγορία και ανά επίπεδο. Οι ηλικιακές κατηγορίες είναι οι παρακάτω, ακολουθώντας πιστά το πρόγραμμα παιδικού Krav Maga της I.K.M.F.: 4-6 ετών (Kiddy), 7-10 ετών (Kids), 11-13 ετών (Junior). Στην αρχή κάθε σεζόν, τα πρόγραμμα διαμορφώνεται με γνώμονα τις ηλικιακές κατηγορίες σε συνδυασμό με τα επίπεδα (αρχάρια/προχωρημένα) των μαθητών.
 Αν ενδιαφέρεστε να έρθετε στη σχολή μας για να γνωρίσετε την αυτοάμυνα Krav Maga από κοντά, σας συστίνουμε να επικοινωνήσετε πρώτα μαζί μας. Οι έμπειροι εκπαιδευτές μας θα σας κατευθύνουν στο κατάλληλο τμήμα για εσάς ή το παιδί σας, έτσι ώστε να έχετε την καλύτερη εμπειρία εκμάθησης.
