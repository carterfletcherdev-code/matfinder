 # Classes

 ### ### -Full schedule of classes ### -Open 6 days a week! ### All are welcome to attend their first class free of charge! Please email, call, or text to schedule your appointment. BJJ All Levels/Fundamentals:
 In this class we cover the basic movements, concepts, and positional hierarchy of Brazilian Jiu Jitsu. This class is an essential place to begin if you are new to the art and all are welcome. These core classes are currently offered 5 times a week.
 BJJ live Drilling:
 This class specifically focuses on core concepts used in BJJ, utilizing repetition as a way to polish essential movements from both top and bottom positions.

 BJJ Intermediate:
 This class explores a deeper understanding of the fundamentals as well as some more situational based strategies and approaches to the game of BJJ.

 Members Open Mat:
 Open Mat is a time specifically set aside for students to practice previously learned techniques in a live setting. There will be no instruction given at these times. These sessions require prior instructor approval.

 Private Lessons/Group Lessons:
 Private and group lessons are available on request to all students regardless of experience or belt rank. Private instruction is a great way to either ease into Jiu Jitsu or to refine specific details in your own Jiu Jitsu game. Please contact Coach Justin Bendsneyder to arrange your session(s).
