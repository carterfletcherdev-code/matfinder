 #### BJJ Fundamentals Class (Ages 13+) Foundations! That’s what this class is all about. The Brazilian Jiu-Jitsu fundamentals class is a perfect way for an adult to start their Brazilian Jiu-Jitsu journey. This class covers all the necessary basics to equip the individual student with the proper tools needed to build their BJJ foundation. The techniques taught in this 1-hour class cover basic positions such as guard, mount, side-control, basic escapes, basic submissions, basic take-downs, and chokes.
 This class is perfect for someone who is brand new to BJJ or for those students who are still focusing on the basics. While you won’t be a beginner for very long, knowing that you can build strength and confidence (both mentally and physically) in a safe, controlled, and regimented learning environment is key to your BJJ future.
