 # KLUB KARATE KONTRA W ŻARACH ### Naszym celem jest propagowanie oryginalnego japońskiego karate, czyli karate shotokan. Organizujemy zawody sportowe, imprezy rekreacyjno-kulturowe, obozy sportowe i szkolenia. Serdecznie zapraszamy dzieci, młodzież i dorosłych. Możliwość lekcji indywidualnych. Zajęcia odbywają się pod okiem Sebastiana Końcowika 4 Dan, instruktora Karate, medalisty Mistrzostw Polski i Europy.
